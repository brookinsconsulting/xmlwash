Installation instructions:

- Either unpack the archive, or get the latest version from SVN.
  URL: http://pubsvn.ez.no/community/trunk/extension/xmlwash


- Place the 'xmlwash' folder in the 'extension' folder.

- Open /settings/override/site.ini.append(.php) and add the xmlwash
  extension to the active extensions.

  [ExtensionSettings]
  ActiveExtensions[]=xmlwash
